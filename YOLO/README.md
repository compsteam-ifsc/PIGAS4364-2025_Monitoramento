# 👀 YOLOv8 Human Detection (Webcam)

This project uses [Ultralytics YOLOv8](https://github.com/ultralytics/ultralytics) with OpenCV to perform **real-time human detection** through your webcam.  
It runs lightweight **YOLOv8n** (nano version) for speed and efficiency.

---

## 🚀 Features
- ✅ Real-time human detection from webcam  
- ✅ YOLOv8 (nano) for faster inference  
- ✅ Bounding boxes with confidence scores  
- ✅ FPS counter  
- ✅ Fullscreen display mode  
- ✅ Exit anytime by pressing **`q`**  

---
